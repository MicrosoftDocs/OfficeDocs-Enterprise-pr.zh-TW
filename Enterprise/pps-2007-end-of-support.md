## <a name="what-does-end-of-support-mean"></a>支援平均數的哪些並結束？

Microsoft 產品，例如 PerformancePoint Server 2007 SP3、 ProClarity 軟體和 SharePoint Server 2007 SP3 所有有支援週期期間 Microsoft 提供的新功能、 錯誤修正程式和安全性更新。 產品生命週期通常會持續 10 年起的產品的初始版本，與該生命週期結束稱為 「 支援的產品的結尾。 為 ProClarity、 PerformancePoint Server 和 SharePoint Server 2007 已到達其結束位置的支援，Microsoft 將不再提供：
  
- 技術支援人員可能發生的問題
    
- 錯誤修正問題，會發現及，可能會影響 exchange 的穩定性與伺服器的可用性
    
- 安全性問題修正所發現和，可能會使伺服器或應用程式容易受到安全性弱點的弱點
    
- 時間的時區更新
    
ProClarity、 SharePoint Server 2007 SP3 和 PerformancePoint Server 2007 SP3 的安裝會繼續執行即使支援結束。 不過，我們強烈建議您儘速移轉從這些應用程式。

### <a name="use-power-bi-in-the-cloud-or-on-premises"></a>使用 Power BI （在雲端或內部）

Power BI 是業務分析工具來分析資料及分享觀點的套件。 使用 Power BI，您可以建立互動式報表和儀表板使用內部部署或線上的資料來源。 使用者可以檢視及使用您的報表和儀表板使用其電腦或行動裝置。
  
Power BI 並不包含在 Office 365 或 SharePoint Server 中，但是會個別提供包含 Power BI Desktop、 Power BI 閘道和 Power BI 服務。 與 SharePoint Online 也整合 power BI。 您可以開始使用 Power BI 免費，並會根據您的資料使用量和業務需求，升級到 Power BI 專業版。 若要深入了解，請參閱[什麼是 Power BI？](https://go.microsoft.com/fwlink/?linkid=841341)
  
### <a name="use-reporting-services-on-premises"></a>使用 Reporting Services （內部）

SQL Server Reporting Services 提供健全的報表解決方案，以及安裝及設定 [原生模式] 或 [SharePoint 整合的模式中的 Reporting Services 的能力。 您可以撰寫使用數種工具，包括報表設計師、 報表產生器和 Power View 報表。 使用 SQL Server 的最新版本，您也可以使用 SQL Server 行動報表發佈者若要擴充的報告傳遞至任何螢幕大小，讓組織能夠使用其行動裝置上的報告。 若要深入了解，請參閱[SQL Server Reporting Services (SSRS): 建立、 部署及管理行動裝置和重新編頁報告](https://go.microsoft.com/fwlink/?linkid=841342)。
