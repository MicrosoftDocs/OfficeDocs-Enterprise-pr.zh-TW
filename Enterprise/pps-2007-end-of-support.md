# <a name="performancepoint-server-2007-end-of-support-roadmap"></a>PerformancePoint Server 2007 終止支援藍圖

Office 2007 伺服器和應用程式已達到其結尾支援，包括伺服器與應用程式，您可能會使用您的商務智慧 (BI) 解決方案的一部分。

## <a name="what-does-end-of-support-mean"></a>支援平均數何種實務結尾？

Microsoft 產品，例如 PerformancePoint Server 2007 SP3、 ProClarity 軟體和 SharePoint Server 2007 SP3 所有有支援週期的期間 Microsoft 提供的新功能、 錯誤修正和安全性更新。產品生命週期通常是一個 10 個年度從產品的初始版本，日期的工期及該生命週期的結尾就所謂的產品支援結束。為 ProClarity、 PerformancePoint Server 及 SharePoint Server 2007 已達到其結尾支援，將不再提供 Microsoft：
  
- 技術支援可能發生的問題
    
- 錯誤修正的問題會發現與之影響的穩定性及可用性的伺服器
    
- 安全性弱點的探索和可能會提出伺服器或應用程式的安全性缺口容易受到的修正
    
- 時間的時區更新
    
安裝的 ProClarity、 SharePoint Server 2007 SP3，與 PerformancePoint Server 2007 SP3 會繼續執行即使支援結束。不過，我們強烈建議您盡移轉從這些應用程式。
