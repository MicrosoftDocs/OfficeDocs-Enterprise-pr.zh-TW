# <a name="move-in-ready"></a>移入就緒

 **摘要：**內建於 Microsoft cloud 方案的儲存選項取得雲端一組的詳細資訊。
  
"移入準備就緒 」 儲存解決方案：
  
- 立即使用預先套裝的解決方案
    
- 已準備就緒，隨附於現有的服務
    
- 提供有限的自訂
    
下列各節說明每個"移入就緒 」 儲存區的詳細資料] 選項。
  
## <a name="sharepoint-online-in-office-365"></a>Office 365 中的 SharePoint Online

### <a name="features"></a>功能

- 檔案且安全、 可靠且可存取來自多個裝置從任何位置
    
- 內容會透過搜尋輕鬆探索
    
- 工作流程與保留原則輕鬆使用及維護的內容
    
- 內容可透過網頁 api （英文） 或瀏覽器中存取或離線
    
### <a name="common-uses"></a>一般用途

- 公司或組織的入口網站
    
- 小組共同
    
- 企業版搜尋
    
### <a name="key-storage-scenarios"></a>主要儲存案例

- 共同作業
    
- 公司通訊
    
- 管理影片
    
### <a name="resources"></a>資源

如需詳細資訊，請按一下[這裡](https://support.office.com/article/SharePoint-Online-and-OneDrive-for-Business-software-boundaries-and-limits-8f34ff47-b749-408b-abc0-b605e1f6d498)。
  
成本資訊，請按一下[這裡](https://products.office.com/business/compare-office-365-for-business-plans)。

## <a name="azure-backup"></a>Azure 備份

### <a name="features"></a>功能

- 順暢地提供保護，從 Windows Server 和 Windows Server Essentials 接收的資料
    
- 包含 System Center Data Protection Manager
    
- 將資料儲存在地理位置複寫的儲存、 跨兩個 Azure 資料中心來維護您資料的六個複本
    
- 支援 delta 變更、 已定義的頻率、 壓縮、 加密、 長保留和頻寬節流設定
