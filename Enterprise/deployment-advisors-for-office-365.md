# <a name="deployment-advisors-for-office-365-or-microsoft-365-services"></a>Office 365 或 Microsoft 365 服務的部署顧問

Office 365 或 Microsoft 365 的部署顧問提供為您的組織設定 Office 365 的自行指導支援。 若要存取這些專案：

1. 在[Microsoft 365 系統管理中心](https://admin.microsoft.com)中以租使用者管理員身分登入。
2. 選取下列其中一個部署顧問或指南。

如果您不是以租使用者管理員身分登入，則會最後看到一則必須具有租使用者管理員許可權的訊息。

每個顧問都是指導您選擇要部署之功能和選項的嚮導。 在某些情況下，您可以使用嚮導來設定內容或啟用設定。 範例如下。

![部署顧問範例](./media/deployment-advisors-for-office-365/m365-deploy-advisor1.png)

Advisor 會逐步引導您完成針對您的需求進行自訂的決策。 順便說一來，您將取得指示、腳本及其他資源，以進行設定變更。

最後，您將取得影片及參考文章的連結。 範例如下。

![影片及參考文章連結的範例](./media/deployment-advisors-for-office-365/m365-deploy-advisor2.png)

您可以隨時使用這些顧問，以深入瞭解 Office 365 的功能和選項。 這包括在規劃期間或您已設定服務之後。
  
## <a name="microsoft-365-windows-10-with-office-365"></a>Microsoft 365： Windows 10 （含 Office 365）

Microsoft 365 是完整的智慧解決方案，包括 Office 365、Windows 10 和 Enterprise 可移動性 + Security （EMS）。 Microsoft 365 讓每個人都有創意，且安全地協同運作。 

使用此 advisor 設定 Windows 10 裝置或將使用者電腦升級至 Windows 10，並選用 Office app、Windows Analytics 和 Windows Defender 高級威脅防護（ATP）的部署（僅限 Microsoft 365 企業版 E5 計畫）。

## <a name="communication-and-online-conferencing"></a>通訊和線上會議

### <a name="office-365-groups"></a>Office 365 群組

Office 365 群組是電子郵件、交談、檔案及事件的共用工作區，讓群組成員可以共同完成工作。 Office 365 中的群組可讓您選擇一組您想要共同作業的人員，並輕鬆地設定要共用的人員資源集合。 您不需要擔心手動指派許可權給所有這些資源，因為新增成員至群組會自動將其所需的許可權提供給群組所提供的工具。

### <a name="microsoft-teams"></a>Microsoft Teams

Microsoft 團隊是 Office 365 中的聊天型共同作業服務，可提供小組工作區，以供聊天、通話、會議及私人郵件。 小組的部署顧問可協助您設定和設定 Exchange Online、SharePoint 線上、商務 OneDrive 與 Office 365 群組的最佳小組體驗。 其包含有關設定專案的承租人設定、小組和管道的資訊、通話和會議、訊息、索引標籤，以快速存取應用程式、資訊 Bot 和連接器至協力廠商服務。
